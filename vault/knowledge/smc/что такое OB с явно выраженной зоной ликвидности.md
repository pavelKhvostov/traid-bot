# Что такое OB с явно выраженной зоной ликвидности

## Идея

«OB с явно выраженной зоной ликвидности» — это обычный canon-OB (пара
`prev`/`cur`), на который **дополнительно наложен маркер ликвидности**.
Маркер не меняет ни геометрию OB, ни его зону входа — это отдельный
**вспомогательный признак** (confluence/контекст), не primitive.

Маркер показывает немитигированную полосу снятой ликвидности: длинный фитиль
`prev`-свечи, в который `cur`-свеча почти не вернулась.

## Условия маркера

Маркер активен при выполнении **всех трёх** условий — тогда строится зона-маркер.
Все условия — про `prev` (OB-свеча с длинным фитилём). Условие фрактала требует
свечу `prev+2` (= `cur+1`), поэтому маркер подтверждается на закрытии `cur+1`.

### SHORT OB

`prev` — бычья, `cur` — медвежья реакция (см. [[универсальные определения OB и FVG]]).

1. **Жёсткое условие выраженности** — верхний фитиль `prev` более чем в 3 раза
   длиннее верхнего фитиля `cur`:
   ```
   (prev.high - max(prev.open, prev.close)) > 3 * (cur.high - max(cur.open, cur.close))
   ```
2. **Условие размера `prev`** — верхний фитиль `prev` больше тела `prev`:
   ```
   (prev.high - max(prev.open, prev.close)) > abs(prev.open - prev.close)
   ```
3. **Условие фрактала** — `prev` является пятисвечным HH-фракталом Билла
   Уильямса (см. [[фракталы билла уильямса]]): `high` строго выше `high`
   двух свечей слева и двух справа:
   ```
   prev.high > high[k]   для k ∈ {prev-2, prev-1, cur, cur+1}
   ```

**Зона-маркер ликвидности**:
```
liq_zone = [cur.high, prev.high]
```
Полоса верхнего фитиля `prev`, торчащая над хаем `cur`.

### LONG OB

`prev` — медвежья, `cur` — бычья реакция.

1. **Жёсткое условие выраженности** — нижний фитиль `prev` более чем в 3 раза
   длиннее нижнего фитиля `cur`:
   ```
   (min(prev.open, prev.close) - prev.low) > 3 * (min(cur.open, cur.close) - cur.low)
   ```
2. **Условие размера `prev`** — нижний фитиль `prev` больше тела `prev`:
   ```
   (min(prev.open, prev.close) - prev.low) > abs(prev.open - prev.close)
   ```
3. **Условие фрактала** — `prev` является пятисвечным LL-фракталом Билла
   Уильямса (см. [[фракталы билла уильямса]]): `low` строго ниже `low`
   двух свечей слева и двух справа:
   ```
   prev.low < low[k]   для k ∈ {prev-2, prev-1, cur, cur+1}
   ```

**Зона-маркер ликвидности**:
```
liq_zone = [prev.low, cur.low]
```
Полоса нижнего фитиля `prev`, торчащая под лоем `cur`.

## Чем НЕ является

- **Не новая зона входа** — зона входа остаётся canon-OB
  ([[универсальные определения OB и FVG]]). Маркер накладывается поверх.
- **Не primitive** — вес вспомогательный (маркер/конфлюенс).
- **Не `SWEPT`** из Strategy 1.1.1: `SWEPT` = OB снял прежний свинг; здесь —
  соотношение фитилей внутри самой пары `(prev, cur)`.

## Статус

Концепт зафиксирован как термин. Детектор и тесты НЕ создаются — это маркер,
не primitive. При необходимости имплементации добавить отдельно.
