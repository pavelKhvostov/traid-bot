# Фракталы Билла Уильямса

## Определение

Свеча `i` — **LL-фрактал** (low-fractal), если:

`low[i] < low[k]` для всех `k ∈ {i-2, i-1, i+1, i+2}`.

Свеча `i` — **HH-фрактал** (high-fractal), если:

`high[i] > high[k]` для всех `k ∈ {i-2, i-1, i+1, i+2}`.

То есть нужно 5 свечей: фракталом является средняя, две слева и две справа должны
её не перекрывать.

## Граничные случаи

В коде ([strategies/fractal.py:10-23](../../../strategies/fractal.py#L10-L23) и
[strategies/hammer.py:17-30](../../../strategies/hammer.py#L17-L30)) проверка строгая:
`<` (а не `<=`). Равные low/high между i и соседом → не фрактал.
