# 3 новых сырых сигнала через единый харнесс: breaker / naked POC / vol-gate ORB

Запрос пользователя: «где деньги — не фильтры поверх книг (потолок), а новые сырые
сигналы». Прогнал 3 неохваченные темы через ОДИН честный харнесс
(`research/daily_engine/zone_harness.py`): лимит/стоп-вход, гонка SL/TP (SL первым
внутри бара), fixed RR=2.2, breakeven WR=31.25%, год-стабильность, R/просадка,
BTC→ETH/SOL, вся история 2020-2026.

## Block 1 — Breaker block (etap_259): KILL
Сломанный OB (close за край) → флип полярности → вход на возврате, SL за зоной.
Детект 4h/12h, исп. 1h. Результат: WR 24–31% (≤ breakeven), R/сделку отрицателен
(4h BTC −0.08, 12h −0.18…−0.23), просадки огромные, год-стаб 0–4/7 — и это на
2000+ сделках (не малый n). Причина: `detect_ob_pair` метит КАЖДУЮ down-up пару →
«сломанных OB» тьма, ноль селективности. Edge 1.1.1 = вложенная мульти-TF конфлюэнция
+ SWEPT, а не сырые OB-реакции. Добавить селективность = пересобрать 1.1.x. KILL.

## Block 2 — Naked POC fade (etap_260): KILL
Незакрытый недельный/дневной volume-POC как зона реакции, fade на первом касании,
SL = ATR-нормированный. Единственная плюсовая ячейка — **ETH дневной** (+0.096 R/сд,
6/7 лет), но НЕ переносится: BTC дневной −0.014 (монетка), SOL −0.087, недельный
слаб/минус везде. Одно-символьный edge = шум (подтверждённая стена «сигналы не
переносятся между монетами»), у ETH 2026 −22.8R. KILL.

## Block 3 — Vol-gate ORB (etap_261): KILL (убит адверсариальной валидацией, etap_261b)
НЕ ставка на направление: IB=3ч UTC, вход В СТОРОНУ ПРОБОЯ (stop), SL=противоп. край IB,
fixed RR2.2. ГЕЙТ: morning eff_ratio≥0.50. Первичный прогон выглядел как кросс-ассет+
net после издержек (BTC +0.098/ETH +0.018/SOL +0.031). Прогнал через адверсариального
агента (7 атак) — **РАЗВАЛИЛСЯ**:
- **Permutation-null (решающее)**: pooled gross +0.0825 vs null «случайный отбор того же
  числа дней из ВСЕХ» = mean +0.050, p=**0.241** → гейт НЕ даёт edge отбора над случайным
  выбором дней. (Coin-null направления p=0.000 — но это тривиальный momentum «вход по
  пробою», НЕ eff-гейт.)
- **Walk-forward** (лучший eff на 2020-23 → applied 2024-26): pooled OOS net **+0.014**,
  BTC OOS **−0.013**, 2 из 3 символов в минус OOS.
- **Threshold sweep** не монотонный, «лучший» порог у каждой монеты свой (0.6/0.7/0.3) —
  0.50 = cherry-pick на рваной n-падающей кривой.
- **RR**: на ETH знак скачет с RR (overfit); **short-сторона** мёртвая на ETH, отрицательная
  на SOL (long-only) — остаток = «BTC long в бычьем режиме», ровно flagged failure mode.
- Lookahead-аудит чист (double-break side-pick меняет 0–1 сделку; time=idx[k-1]+stop без
  утечки; SL-first пессимистичен).
Вердикт: **KILL**. Единственное, что выжило — тривиальный факт «вход по направлению пробоя
бьёт монетку» (momentum), а не vol-гейт как механизм отбора.

## Уроки (в known-pitfalls)
- ORB/узкий-стоп: издержки в R ВЕЛИКИ относительно стопа — всегда cost-adjust в R
  (cost_R = round-trip% / risk%), иначе тонкий gross-edge мнимый. (Здесь стопы оказались
  шире ожидаемого, edge выжил — но проверять ОБЯЗАТЕЛЬНО.)
- Сырой SMC-детектор без селективности (каждый OB-пар) = ноль edge; edge в КОНФЛЮЭНЦИИ.
- Одно-символьный плюс (ETH) при монетке на BTC/SOL = шум (повтор стены).
