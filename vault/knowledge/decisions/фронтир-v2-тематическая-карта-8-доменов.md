# Фронтир модели анализа v2.0 — тематическая карта 8 доменов

Workflow `wf_5db33fd8-491` (8 разведчиков + 8 адверсариальных верификаторов;
все внешние источники **проверены живыми запросами через системный прокси**).
Полный дамп: `research/daily_engine/output/etap_242_frontier_full.json`.
Контекст: ICT/SMC выжат, направление = стена; бьём в разрешённые таргеты
(range R²0.50 / big_day 0.73 / trend-hold / choppiness 0.631 / ячейки / продукт).

## Ключевые открытия разведки (данные)

- **Binance Futures доступен глубоко**: REST fundingRate с 2019-09 (BTC);
  premiumIndexKlines с 2019-12; **S3-дампы data.binance.vision futures/um/daily/metrics**
  = 5-мин CSV с open_interest + 4 long/short ratio, BTC с 2020-09, ETH/SOL с 2021-12.
  (REST openInterestHist — правда только 30 дней; дампы решают.)
- **Ликвидации исторически НЕдоступны** (дампы um прекращены 2024-03, Coinglass free
  не существует) → только прокси через 5-мин ΔOI-спайки; cm-снапшоты 2023+ для валидации.
- **Deribit public API работает через наш прокси без ключа**: DVOL (крипто-VIX)
  daily+hourly BTC/ETH с 2021-03-24. SOL DVOL мёртв → VOLMEX:SVIV14D через TV CLI.
- FOMC/CPI/NFP даты = детерминированная фича будущего без lookahead.

## План исполнения (3 волны)

**Волна 1 — локальные данные, 0 внешних рисков (1 связка скриптов):**
1. `etap_243`: дневной vol-датасет из 1m (HAR-RV + BNS-jumps + semivariance/skew
   + RV-кривая по часам) → A/B против range/big_day базы (etap_204 walk-forward).
2. `etap_244`: MultiQuantile + CQR на лучшем наборе из 243 → распределение дня.
3. `etap_245`: choppiness/trend-hold bake-off (VR-профиль + ER + run-length утра
   против текущих n_cross/dsum).

**Волна 2 — внешние проверенные источники:**
4. `etap_246`: граббер funding + S3 metrics (один скрипт, кэш CSV) →
   derivs-2 квадранты ΔOI → trend-hold; derivs-1 funding-фичи → big_day/range.
5. `etap_247`: DVOL граббер → options-1 prior в range-модель + options-5
   IV expected move в продукт (границы дня в дашборд).

**Волна 3 — календарный пакет:**
6. `etap_248`: FOMC/CPI/NFP + экспирации + turn-of-month + US-holidays →
   единый календарный фичесет → A/B big_day/range; calendar-4 сессионная
   декомпозиция остатка дня.

Каждый этап — с kill-критерием из вердиктов (полные спецификации в дампе).
