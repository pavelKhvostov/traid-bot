# ICT double-FVG: формации по контексту (12h–3d, BTC+ETH)

etap_253 (формации) + etap_254 (контекст). Канон FVG c1-c3, без лукахеда:
bull c1.high<c3.low, bear c1.low>c3.high. Double = два подряд FVG одного направления
(displacement-leg). Forward 5 баров: dir_next / cont / fill(митигация) / fwd_range×.

## Главное: double-FVG — НЕ одно явление, контекст решает

**Направление = монетка во ВСЕХ контекстах** (dir_next 40–53%, год-нестабильно) —
стена держится. НО продолжение/заполнение/магнитуда расходятся резко по контексту:

| Контекст (пул BTC+ETH, 12h-3d, n=744) | n | продолж. | гэп заполнен | ход× |
|---|---|---|---|---|
| **Continuation-HQ** (по тренду EMA50 + смещение ≥2.9 ATR) | 351 | **61%** | 49% (держится) | **3.14×** |
| против тренда / слабое | 392 | 52% | 69% | 2.30× |
| худшее (против И слабое) | 62 | 42% | **77%** (фейл) | 2.05× |

Отдельные оси (год-проверены):
- **Локация**: премиум (верх 20-бар диапазона) cont 64% / ход 2.95×; дисконт cont 44%.
  Контринтуитивно для наивного ICT, но логично: double в премиуме = тренд уже едет.
- **Против тренда**: fill 78%, cont 38% → ловушка, гэп быстро закрывается.
- **Сила смещения**: сильное (≥2.9 ATR) гэп держит (fill 50%, ход 3.1×); слабое fill 69%.
- **Импульс до**: «после движения в сторону FVG» — единственный год-стабильный (71% лет)
  directional намёк, но 53% next-bar (слабо, не торгуемо).

## Базовые forward-стат (etap_253, double vs single, все ТФ)
fwd_range× double ~2.8–3.0 vs single ~2.4 (расширение хода — робастно);
fill double НИЖЕ single (BTC-3d 39% vs 67%) — имбаланс держится; cont на 2d/3d 56–67%.

## Практическое правило
Double-FVG = **топливо ИЛИ ловушка** по контексту:
- по тренду + сильное смещение → FVG = реальная опора-топливо, лин на продолжение
  (размер от зон/типа дня, НЕ ставка на направление);
- против тренда / слабое → гэп заполнится (фейд-контекст), не гнаться.

## Как ТОРГУЕМЫЙ сигнал (etap_264/264b, 2026-06-18) — KILL
Операционализировал continuation-HQ как сделку (#2 из PDF-стратегий): double-FVG + тренд(EMA50)
+ displacement>=2.9ATR -> лимит-вход в FVG, SL за ним, fixed RR. Первый прогон выглядел кросс-
ассет+ (BTC +0.385 RR2.5, ETH +0.143, SOL +0.278), НО:
- **Permutation-null (etap_264b): HQ-фильтр (disp>=2.9+тренд) НЕ бьёт случайный отбор** того же
  числа double-FVG (p=0.10–0.67 на всех RR/ассетах; HQ-подвыборка внутри нуля). Год-стаб-«находка»
  как СЕЛЕКТОР = мираж.
- **Остаточный плюс базового double-FVG = bull-drift**: LONG +0.15..+0.38 / SHORT ~0..−0.05 на
  всех 3 монетах. Положителен только лонг в 6-летнем бычьем рынке, не сетап-edge.
→ Continuation-HQ остаётся ОПИСАТЕЛЬНОЙ находкой, но НЕ торгуемый сигнал. Та же стена:
направление=монетка/дрейф, фильтр не бьёт случай.
