# Фрактальный ±5% first-passage: направление = монетка; модуль САМ поймал свой мираж

## Задача (юзер)
Самообучаемый, само-исправляющийся нейро-модуль: на закрытии 12h-свечи (цена P) аргументированно,
используя все знания/индикаторы/поток, определить — станет ли свеча ФРАКТАЛОМ и даст ли +5% (P·1.05)
РАНЬШЕ, чем −5% (P·0.95). Т.е. close 50000 → 52500 раньше 47500? И модуль ДОЛЖЕН осознать ошибку,
понять причину, исправиться аргументированно (тот же запрос на само-исправление, что в zone-race).

## Что это математически
Направленный **first-passage на СИММЕТРИЧНЫХ барьерах ±5%**. Дистанция = константа (оба 5%) →
gambler's ruin даёт ровно **50%** (driftless). Любой edge обязан идти от drift-нейтральной СТРУКТУРЫ.
Фрактал Williams k=2 подтверждается через i+2 → LOOKAHEAD → только в МЕТКУ, никогда не фича.

Код: `research/ta_laws/fractal_first_passage.py` (+ `_report.txt`, `.png`). 13 464 якоря BTC/ETH/SOL,
горизонт 30д, путь ±5% по 1h строго после close 12h-свечи, 23 каузальные фичи (моментум/вола/поток
CVD-delta-taker/EMA-структура/HTF/инверсия-флага/тело-фитили). OnlineLaw (L1) + CatBoost walk-forward (L2).

## Ключевой момент — модуль поймал СВОЙ мираж (это и есть само-исправление)
- **L1 онлайн «нейро»: 67.8%, «учится» 64.5%→71.1%** — выглядело прорывом.
- **L2 block-OOS AUC: 0.51** — монетка. ПРОТИВОРЕЧИЕ.
- Подозрение по нашей стене (acc≥65% на направленной крипте → искать lookahead/autocorr) запустило L1.5:
  - lag-1 автокорр метки y: **+0.72 / +0.62 / +0.48** (BTC/ETH/SOL); средн. длина серии ~5 «смен».
  - baseline ПЕРСИСТЕНТНОСТИ `pred[i]=y[i-1]` («как в прошлый раз») = **80.6%** ← вот источник «точности».
  - OnlineLaw на ПЕРЕТАСОВАННОМ времени = **50.5%** ~ маж. 51.4% (автокорр разорвана → сигнал исчез).
  - + утечка: метку y[i-1] узнаём лишь через 30д после close[i-1] = на ~30д ПОЗЖЕ close[i]. Онлайн
    обучался на ещё-не-зарезолвленных (будущих) метках = 30-дн. label-lookahead.
- **ОСОЗНАНИЕ:** онлайн-68% = персистентность сильно-автокоррелированной метки + утечка, НЕ предсказание.
  Исправление: доверять ТОЛЬКО block-OOS.

## Честный вердикт (block-OOS, drift-aware)
- OOS AUC **0.510** (perm-null p=0.050 — ровно на грани), но только **2/6 лет >0.52** (2021/23/26 < 0.5).
- cross-asset leave-one-asset 0.541/0.541/0.531 — шёпот, но LOA оптимистичен (нет тайм-пёрджа, все годы).
- дрейф-раскол: up-вызовы 51.1% (база 51.4), down-вызовы 50.6% (база 48.6) — симметрично, остаток = дрейф.
- drift-нейтрально (баланс классов по годам): AUC **0.5143**.
- НЕТТО (±5% брекет RR1, кост 0.14%RT≈0.028R): WR 51.4%, netExp **+0.001R/сделку** = ноль.
- **ИТОГ: направление ±5% от 12h-свечи = монетка (50%) + бычий дрейф (51.4%). Не торгуемо сверх дрейфа.**

## Описательно (fr_dir подтверждён через i±2 — НЕ торгуемо)
dn-фрактал (свинг-лоу) → up-first **70.7%**; up-фрактал → 34.3%; не фрактал → 51.0%. Т.е. ПОДТВЕРЖДЁННЫЙ
свинг действительно «дал свои 5%» в свою сторону — но это известно лишь через 2 бара (почти тавтология
свинг=локальный экстремум), это валидация структуры, не предсказание.
