# Нейро-модуль «законов ТА»: континуация флага/клина/вымпела = ФОЛЬКЛОР

Запрос юзера: добавить к аналитике «нейро-модуль», который ищет паттерны, а я вывожу из них
ЗАКОНЫ ТА. Дал 7 графиков-аннотаций: даунтренд → восходящая коррекция (канал/клин/вымпел) →
стрелка вниз к measured-move TP (continuation). Построен модуль: geometry → labeler → validator.

## Модуль (research/ta_laws/)
- `geometry.py` — zigzag-пивоты (+ conf_i для каузальности) → импульс (флагшток) → классификатор
  коррекции (CHANNEL/FLAG/RISING_WEDGE/FALLING_WEDGE/PENNANT/ASC_TRI/DESC_TRI) + measured-move.
  Тесты `tests/test_ta_geometry.py` 7/7. Визуал `plot_patterns.py` (сверено с графиками юзера — формы совпали).
- `labeler.py` — каузально (arm=conf последнего пивота) размечает КАЖДЫЙ архетип: tradeable measured-move
  (пробой по СПРОЕЦИРОВАННЫМ линиям → TP/стоп) + **bias-free signed forward-return** + null (случайные бары
  той же дирекшн-смеси/горизонта) + режим (трейлинг BTC) + HTF-выравнивание.
- `validate_laws.py` — гейты: fwd>0 И permutation-null p<0.05 И ≥2/3 символов+ И оба режима+.

## Вердикт (4671 архетип BTC/ETH/SOL): 0 законов, всё ФОЛЬКЛОР
- ВСЕ: fwd_signed **−0.0055**, p=1.000 (хуже null −0.0009), 0/3 символов+, оба режима−. expR −0.27.
- **against=1 (ровно сетап с 7 фото — флаг/клин ПРОТИВ импульса): fwd −0.0125, WR 8%, expR −0.69 — ХУДШИЙ.**
- FLAG fwd −0.016 WR 15%; PENNANT −0.006; RISING_WEDGE WR 10%. measured-move сделка: 629 вошло,
  WR 32%, RR 7.95, **ΣR −170R**. Воронка: **reversed 3204/4671 (69%)** — паттерны чаще ломаются ПРОТИВ.
- Симметрия: UP −0.0034 / DOWN −0.0076 (оба−) → не «медвежья бета», а реальная анти-континуация.
- Единственное не-убыточное: htf-aligned p=0.523 (≈ random, не edge).

## Смысл
7 графиков юзера = **иллюзия отбора** (нарисованы в точке пробоя со стрелкой-прогнозом). Объективно
голая геометрия континуации НЕ работает (две независимые метрики: fwd-return + null). Согласуется со
стеной проекта: направление=монетка, паттерны не предсказывают, reversal — трудная сторона. Работающий
edge проекта = реакция у КОНФЛЮЭНТНЫХ мульти-TF зон (каскады 1.1.x), а не форма.

## Зацепки на будущее (НЕ подтверждено)
- **Инверсия**: against=1 даёт −0.0125 в обоих режимах/всех символах → «флаг ломается ПО коррекции
  (= разворот), а не по импульсу» может быть реальным законом. Прогнать через те же гейты (вдруг дрейф/измерение).
- **Условные законы**: голая форма мертва, но форма + конфлюэнтная HTF-зона + сжатие объёма + order-flow
  на пробое — вот где искать. Модуль = майнер УСЛОВИЙ, не форм.

## ВАЛИДИРОВАННЫЙ ЗАКОН (инверсия) — 1h→D, с 2020, 7653 архетипа

Расширили на ВСЕ ТФ 1h→1d с 2020 + метрика triple-barrier FADE (±1.5 ATR, что первым: against
или continuation) + pivot-null контроль. **FADE прошёл ВСЕ 21 срез гейтов.**

**ЗАКОН (инверсия импульса-коррекции / «fade the flag»):** после импульса контр-структурная
коррекция (флаг/клин/вымпел/канал) чаще ПРОДОЛЖАЕТ КОРРЕКЦИЮ (разворачивает импульс), чем пробивает
по импульсу. ~63% бьют 1.5-ATR анти-импульс барьер первым. fade expR **+0.252**, p=0.000.

Гейты (все пройдены): cross-asset 3/3, оба режима (bull/bear), обе стороны (UP+DOWN),
**ВСЕ ТФ 1h/2h/4h/6h/12h/1d**, **год-стабильность 7/7** (и для against=1: 7/7, +0.40..+0.55).

**Pivot-null контроль (решающий):** generic swing-continuation на любом пивоте = +0.036 (≈0);
архетипы +0.252 → **прирост паттерна +0.216 (6×)** → это ПАТТЕРН, не тривиальный пост-пивот моментум.

**Условия-усилители** (actionable): against=1 +0.45 · FLAG +0.52 · импульс ПРОТИВ HTF +0.47 ·
мелкая/средняя глубина +0.36/+0.39 · слабый импульс <4ATR +0.37. (по HTF +0.12 — слабее).

**Зеркало:** текстбук-континуация (measured-move по импульсу) = ФОЛЬКЛОР: 1039 вошло, WR 35%,
expR −0.20, ΣR −212R, reversed 5251/7653 (69%). Т.е. 7 графиков юзера «вниз по measured-move» —
проигрышная сторона; выигрышная = fade (коррекция продолжается).

Кавеат: expR на симметричном ±1.5ATR барьере (не реальная сделка с проскальзыванием/фандингом);
направленный закон твёрдый, монетизация (entry/stop/exit + costs) — отдельный шаг.
