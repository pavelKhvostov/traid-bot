# Законы фигур ТА: КУДА раскроется + ДОКУДА дойдёт

Модуль `research/ta_laws/figures.py` (DT/DB, H&S/inverse, triple, треугольники, прямоугольник)
+ `figure_analysis.py`. Каузально (arm=conf завершающего пивота). 42595 фигур, BTC/ETH/SOL, 1h→1d, 2020-26.
Тесты `tests/test_ta_figures.py` 6/6.

## ЗАКОН 1 — НАПРАВЛЕНИЕ (куда раскроется)
Метрика: triple-barrier ±1.5ATR, signed by учебной стороной; P(учеб первой), vs null (null +0.012).

**Реверсивные фигуры ломаются ПО учебнику — но СЛАБО и только в АЛИГНЕ с трендом:**
- INV_HEAD_SHOULDERS → UP: P 55.8%, +0.116, p=0.001, 3/3 (сильнейшая)
- DOUBLE_BOTTOM → UP: 53.0%, +0.060 · DOUBLE_TOP → DOWN: 52.6%, +0.052 (p=0, 3/3)
- TRIPLE_TOP → DOWN 54%, +0.081 · TRIPLE_BOTTOM → UP 53%, +0.064 (p<0.05, 3/3)
- HEAD_SHOULDERS → DOWN: +0.026 p=0.337 — **НЕ значим** (вопреки учебнику)

**Континуационные фигуры = монетка или ОПРОВЕРГНУТЫ:**
- ASC_TRIANGLE: учебник UP, реально **−0.057 (ломается ВНИЗ), p=1.0** — фольклор!
- DESC_TRIANGLE p=0.614 · SYM_TRIANGLE p=0.306 · RECTANGLE p=0.907 — монетки.

**Ключевое условие НАПРАВЛЕНИЯ:** edge есть только когда фигура АЛИГНА с режимом/трендом.
Пример DOUBLE_TOP: aligned(по режиму) +0.11, против режима +0.01. Т.е. «реверсивные» фигуры
работают как ПРОДОЛЖЕНИЕ старшего тренда (DB в аптренде), а не как контр-тренд разворот.
Up-фигуры сильнее down (крипто-дрейф — кавеат). Магнитуда мала (52-56%): реально, но слабо.

Контраст с fade-законом: тот fade требовал ПРЕДШЕСТВУЮЩЕГО импульса; здесь standalone-консолидации
(треугольники/прямоугольник) без импульса = монетки. → FADE-edge привязан к импульсу, не к форме самой.

## ЗАКОН 2 — ЦЕЛЬ (докуда дойдёт), MFE в «высотах фигуры», при подтверждённом пробое
- **Медиана хода = 0.49× высоты фигуры** (учебный measured-move 1× — ЗАВЫШЕН). Лишь **29% доходят до 1×**, 12% до 2×.
- По фигурам: DT/DB лучший follow-through 0.66-0.68× (37% до 1×); H&S/RECT ~0.5×; треугольники худшие 0.31-0.33×.

## ЗАКОН 3 — ОТ ЧЕГО ЗАВИСИТ ЦЕЛЬ (факторы)
- **РАЗМЕР фигуры — главный фактор (обратный!):** маленькая <3ATR → **0.77×** (41% до 1×);
  средняя 3-6ATR → 0.48×; большая ≥6ATR → **0.24×** (14%). Маленькие фигуры добивают цель НАМНОГО надёжнее.
- **Импульс бара-пробоя:** сильный (≥1.5ATR) → 0.57× (34% до 1×) vs слабый 0.45× (27%).
- **Алигн со старшим трендом / режимом:** +0.05-0.06× к медиане (скромно но стабильно).
- **ТФ-инвариантно**: 0.43-0.50× на всех 1h→1d (ход в высотах не зависит от ТФ).

## Практический свод (на что опираться)
1. Из «фигур» направленный edge есть только у РЕВЕРСИВНЫХ (DT/DB/triple/iHS) и только В АЛИГНЕ с трендом; слабый.
2. Континуационные (треугольники/прямоугольник/H&S) — НЕ доверять направлению (монетка/фольклор; ASC_TRI ломается вниз).
3. Цель ставить на **~0.5× высоты**, не 1×. Полный measured-move = ~30% вероятность.
4. Лучшие условия добить цель: **маленькая фигура + сильный бар-пробой + по тренду**.
5. Самый сильный edge проекта = не «фигура», а fade импульс-коррекции ([[ta-chart-pattern-continuation-folklore]]) + каскады.
