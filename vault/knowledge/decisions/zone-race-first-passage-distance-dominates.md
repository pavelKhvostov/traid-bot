# Нейро-модуль «гонка зон»: в какую зону первой = ДИСТАНЦИЯ (~73%), зона-сила не добавляет

Запрос: построить само-обучающийся/само-исправляющийся нейро-модуль, отвечающий «в какую зону Вадима цена
придёт ПЕРВОЙ», который при ошибке ОСОЗНАЁТ её, понимает причину и аргументированно исправляется.
`research/ta_laws/zone_race_module.py`. Переформулировка неразрешимого «куда пойдёт» → разрешимое first-passage.

## Построено (архитектура работает)
- Гонка: на каждом якоре ближайшая незакрытая зона ↑/↓ (канон-движок Вадима), метка first-passage каузально
  (форвард 1h, горизонт 60h), 11029 якорей BTC/ETH/SOL (каждые 12ч, зоны 12h/1d).
- **Онлайн-предиктор (walk-forward, без lookahead) с само-исправлением**: предсказал → исход → осознал ошибку →
  АТРИБУТИРОВАЛ причину (фича с макс. вкладом в неверную сторону) → АРГУМЕНТИРОВАННО сдвинул её вес (онлайн-
  градиент) + лог довода. Running-нормализация каузально. Каждый якорь ревизует веса = пересмотр прошлых выводов.

## Результат (метрика без AUC — точность)
- Монетка 50%; **baseline «ближайшая зона первой» (геометрия) = 73.1%**; **нейро-модуль 72.7% (−0.4 п.п.)**.
- Самообучение: ранняя 72.4% → поздняя 73.0% (учится, сходится к дистанции). Cross-asset 73/73/71% стабильно.
- Лог само-коррекции живой (пример: «предсказал ВНИЗ p=0.13, факт ВВЕРХ; причина dist_ratio; w −0.088»).
- Частота причин ошибок: dist_ratio 1790 ≫ остальных. Итоговые веса: **dist_ratio +1.55 ДОМИНИРУЕТ**, str/mag/mtf ≈0.

## Закон (выведен)
1. **First-passage к зоне ≈ ДИСТАНЦИЯ**: ближайшая зона приходит первой ~73% (cross-asset 3/3, каузально) —
   настоящий робастный описательный закон (почти тривиальный, но реальный).
2. **Зоны Вадима = КАРТА кандидатов-целей, но их сила/роль/магнит/контекст НЕ добавляют сверх дистанции**
   (72.7% ≤ 73.1%). Модуль это САМ осознал и аргументированно прижал их веса к 0 — требование «понять ошибку и
   исправиться обоснованно» выполнено буквально; честный вывод модуля = «доверяй дистанции».
3. Согласуется со всей линией: зоны = карта+фильтр, не предсказатель; «куда первой» решает геометрия, не зона-магия.

## v2 — МОМЕНТУМ тоже не бьёт дистанцию (расследование исчерпано)
`zone_race_v2.py`: добавлены velocity_3/6/12 + accel (самый сильный априорный кандидат — дрейф в first-passage)
+ range_pos. Решающая стратификация по |dist_ratio|: даже в РАВНОУДАЛЁННОМ режиме (где геометрия бесполезна)
нейро 56.7% vs baseline 57.1% (−0.4); корреляции скоростей с исходом ≈0 (vel3 +0.006, vel6 +0.012, vel12 +0.032,
accel −0.033); dist_ratio +0.178 всё равно сильнейшая, хвост mtf +0.09. Веса: dist_ratio +1.69 ≫, скорости≈0.
**Итог: проверены и отвергнуты сверх дистанции — зона-сила/магнит/роль (v1), мульти-ТФ контекст, моментум/
скорость/ускорение (v2). First-passage = ЧИСТАЯ ГЕОМЕТРИЯ.** Рынок не телеграфирует «какую зону первой» сверх «ближайшую».

## КОНТРОЛЬ (возражение юзера: «это же gambler's ruin, не закон»): ПОДТВЕРЖДЕНО — чистая геометрия
`zone_race_control2.py`. (1) up-first 51% → симметрично, направления нет (ближняя зона попеременно ↑/↓,
поэтому «ближайшая первой» ≠ «рынок всегда рос/падал»). (2) **Калибровка gambler's ruin почти идеальна**:
формула d↓/(d↑+d↓) vs факт по бакетам — 13/16, 34/33, 51/51, 69/70, 88/86% → first-passage = DRIFTLESS геометрия.
(3) **Зоны Вадима 71.9% vs random на ТЕХ ЖЕ дистанциях 72.2% (−0.4пп) → ЗОНЫ ≈ СЛУЧАЙНЫЕ УРОВНИ.**
КАВЕАТ-урок: контроль-1 дал ложные +3.1пп (random брал на РАВНОМЕРНЫХ дистанциях ≠ распределению зон —
distance-distribution артефакт); честный контроль = random на тех же дистанциях. **ВЫВОД: «какая зона первой» =
математическая теорема (gambler's ruin), НЕ закон рынка; зоны для гонки к касанию не несут инфо. Касание =
геометрия; ценность зон — в РЕАКЦИИ ПОСЛЕ касания (фильтр магнит/clear-path ✅, вход-в-зону Дв.2 KILL ❌).**
Вопрос «какая зона первой» геометрически плохо поставлен для проверки ценности зон.

## КАЛИБРОВКА first-passage (рабочий артефакт — но это МАТЕМАТИКА, не зоны)
P(ближняя зона первой) по асимметрии дистанции: явно ближе → **88%**, умеренно → **73%**, почти равноудал. → **57%**.
Кросс-ассет, каузально. Применимо к брекету: калиброванная вероятность, какой уровень (TP↑ vs стоп↓ у зон) сработает первым.

## Применимость
Не прогноз направления, а **риск-инструмент брекета**: из двух уровней (TP↑ vs стоп↓ у зон) ближайший
срабатывает первым ~73% → подсказка по выбору цели/стопа и сайзингу. Само-исправляющийся каркас готов принять
новые фичи, если найдём бьющие дистанцию (velocity/momentum к зоне, order-flow — кандидаты).
