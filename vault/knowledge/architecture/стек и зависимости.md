# Стек и зависимости

## Среда

- Python 3.13 (см. shebang/CLAUDE.md)
- Зависимости в `requirements.txt` — **без зафиксированных версий** (см.
  [[технический долг апрель 2026]] пункт 2).

## Ключевые библиотеки

| Пакет | Зачем |
|---|---|
| `pandas` | OHLCV, `compose_from_base` через `resample(origin="epoch")`, см. [[pandas-frequency-lowercase]] |
| `requests` | sync HTTP в `telegram_bot.py` и `data_manager.py` (см. [[технический долг апрель 2026]] пункт 3) |
| `websockets` | live-поток Binance в `scanner.ws_loop` |
| `python-dotenv` | `load_dotenv()` в `config.py` для `TELEGRAM_BOT_TOKEN` |

## Чего **нет** (и не будет на MVP)

- ORM / Postgres / SQLite ([[почему csv а не postgres]])
- TA-Lib / pandas-ta — индикаторов в проекте нет, вся логика в детекторах
- Docker — бот запускается одной командой `python main.py`
- async HTTP (`httpx`/`aiohttp`) — пункт техдолга
- `pytest` — тестов нет, только `smoke_test*.py` руками
- python-telegram-bot — голый Telegram Bot API через `requests`
