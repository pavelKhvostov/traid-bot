# Сессия 2026-06-11: команда агентов + TradingView MCP + разметка графика

## Что сделано (хронология)

### 1. Витрина сигналов (etap_183)
Прогнал каждый сигнал каждой стратегии+фрактала через нейросеть (честная OOS-оценка
1-5) + реальный исход. CSV `output/etap183_signal_showcase.csv` (9904 сигнала).
Результат: монотонность работает (NN 2→3→4 = 29→32→38% реально хороших), но фильтр
умеренный (lift ×1.17). По годам: прибыльно но нестабильно, 2022 убыточен.

### 2. ⚠️ ИСПРАВЛЕНА КРУПНАЯ ОШИБКА — WR стратегий
Считал WR на ГОЛЫХ зонах без индикаторов → занижал (1.1.1 = 35% вместо реальных 51.9%).
Запустил эталонный etap_103_floating_tp.py: РЕАЛЬНЫЙ WR 1.1.1 floating =
BTC 51.9%/ETH 53.2%/SOL 45.3% (4-indicator score Hull/MH/RSI/ASVK). Также сказал
«риск не продуман» когда всё зафиксировано через 3 этапа (entry 0.80 → SL 0.35 → RR 2.2).
ВЫВОД: стратегии с индикаторами (50-67%) СИЛЬНЕЕ нейросети-оценщика (37%).

### 3. etap_185 — нейросеть на разворотах Вадима (отрицательный)

АРТЕФАКТ. Нейросеть НЕ усиливает Вадима (мало данных + уже отфильтровано).

## Главные выводы для следующей сессии

1. **Нейросеть-ветка исследована до потолка.** Работает для предсказания РАЗВОРОТА
   (etap_177 AUC 0.93), НЕ работает для оценки исхода (Bayes error) и фильтра поверх
   готовых сигналов (артефакт). Реальная ценность = стратегии с индикаторами + ViC Vadim 78%.
2. **TradingView MCP работает** — можно размечать графики, сверять индикаторы.
3. **Команда из 7 агентов** не даст повторить казусы (WR без индикаторов, артефакты).
