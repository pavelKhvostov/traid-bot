## Новые ошибки в журнале

### #18 — Per-row scope ±2% вместо ±20%
Перепутал context scope (±2%) с per-row scope (±20%). User помнил v3 baseline = 18 OB на 4h, мой smoke давал 3.
**Fix:** PER_ROW_SCOPE_PCT = 20.0, CONTEXT_SCOPE_PCT = 2.0 раздельно.

### #19 — Invalid active_zone в simplified LTF path
В `_wick_fill_simplified` (15m/30m): когда фитиль перепрыгивал через границу (LONG: low ≤ z_lo, SHORT: high ≥ z_hi), эмитился fill_partial с **invalid bounds** (a_lo > a_hi). 172,918 events (42.9%) в v10 были corrupted.
**Fix:** в simplified path фильтр `fill_idx < retire_idx`. Sanity assertion в snapshot.
