# Сессия: цепочки A/B/C → грид каскадов → декорреляция → нейро-модуль законов ТА

Очень длинная сессия. Две большие линии: (1) расширение торговых цепочек (cross-asset, грид,
декоррелированная корзина для фьючерсов); (2) построение «нейро-модуля законов ТА» — от геометрии
паттернов до выведенных и фальсифицированных законов. Сквозная дисциплina: «закон/edge только если
выживает permutation-null + cross-asset + год + режим + (pivot-)null», иначе ФОЛЬКЛОР.

## 1. Три цепочки A/B/C через cross-asset гейт
- **A i-RDRB+FVG (Combined-D) → ✅ ЖИВАЯ live-кандидат №1**: детектор `strategies/strategy_i_rdrb_fvg.py`
  + 8 тестов + cross-asset бэктест. 1h: BTC +127R/6/7лет, ETH +96R, SOL маргинал; двусторонняя
  (не bull-drift); воспроизводит Combined-D. [[i-rdrb-fvg-cross-asset-live-candidate]]
- **B 1.1.7 iFVG → BTC-only**: cross-asset гейт (etap_95 V2c полная история) — ETH −29.5R/2/7, SOL=артефакт 2022. [[strategy-1-1-7-ifvg-continuation]]
- **C RDRB-htf → грейд-бустер, не замена**: avg-R выше, но в 3.7× реже → total R ниже. [[rdrb-htf-1-1-1-high-conviction-subset]]

## 2. Ночной грид 64 каскадов
`research/cascade_grid/grid.py` (Stage1 гейты) + `stage2_validate.py` (адверсарно null2000). **12/64 робастны
cross-asset.** Грид сам переоткрыл all-OB=1.1.2 как универсал. Строгий live-скелет (`live_skeleton_top3.py`)
дал **новый live-кандидат cand2 = FVG-якорь** (top FVG-1d/12h + OB-macro→OB-htf, per-trade +0.43/+0.33/+0.24).
Кавеат: грид-ΣR завышен (PER_TOP_CAP) → ранжировать по per-trade+null. [[cascade-grid-64-cross-asset-robust-shortlist]]

## 3. Декоррелированная корзина для фьючерсов
Юзер: нужны РАЗНЫЕ цепочки (не дубли) для независимых фьюч-позиций. `research/cascade_grid/decorrelation.py`:
overlap(Jaccard) макс 0.19, корреляция месячных R макс 0.31, есть ОТРИЦАТЕЛЬНЫЕ (1.1.5↔1.1.1 −0.27,
A↔OB-каскады −0.1..−0.15). Диверсификация Sharpe 0.55→0.88. **Корзина: A + 1.1.2 + cand2(FVG-якорь) +
1.1.5 + 3.2**; cand1-htfRDRB→грейд на 1.1.2. Кавеат: ΣR=база-геометрия. Баг по пути: воркфлоу-аудит упал
(субагенты 403 в headless) → аудит solo; 1.1.5/1.1.6 KeyError починены. [[decorrelated-chain-basket-for-futures]]

## 4. НЕЙРО-МОДУЛЬ ЗАКОНОВ ТА (главное) — `research/ta_laws/`
Запрос: модуль ищет паттерны, я вывожу ЗАКОНЫ. Дал 7 графиков (импульс→флаг/клин→континуация).
- **geometry.py** — zigzag(+conf_i каузальность)→импульс→классификатор коррекции (CHANNEL/FLAG/WEDGE/
  PENNANT/TRIANGLE) + measured-move. Тесты 7/7. Сверено визуально — формы совпали с графиками юзера.
- **figures.py** — DT/DB, H&S/inverse, triple, треугольники, прямоугольник. Тесты 6/6.
- **labeler.py** — каузально (arm=conf), triple-barrier fade + bias-free fwd-return + null + pivot-null + факторы.
- **validate_laws.py / figure_analysis.py / factor_rank.py** — гейты + ранжирование.

### Выведенные законы
- **Континуация = ФОЛЬКЛОР**: «импульс→флаг→продолжение» fwd −0.0055, 69% reversed, measured-move сделка
  WR 35% / −212R. 7 графиков юзера = иллюзия отбора (рисуются в точке пробоя). [[ta-chart-pattern-continuation-folklore]]
- **✅ ЗАКОН ИНВЕРСИИ «fade the flag»** (1h→D, 2020-26, 7653 архетипа): после импульса коррекция чаще
  ПРОДОЛЖАЕТ коррекцию, чем пробивает по импульсу (~63%, fade expR +0.252). ВСЕ 21 срез гейтов:
  cross-asset 3/3, оба режима, UP+DOWN, все ТФ, **год 7/7**. **Pivot-null контроль решающий**: generic
  momentum +0.036 vs архетип +0.252 → прирост паттерна +0.216 (6×) = это ПАТТЕРН, не моментум.
- **Законы фигур** (42595 фигур): реверсивные (DT/DB/triple/iHS) ломаются по учебнику но СЛАБО и только в
  алигне с трендом; континуационные = монетка, **ASC_TRIANGLE опровергнут (ломается вниз)**. ЦЕЛЬ:
  медиана **0.49× высоты** (учебный 1× завышен), фактор размера обратный (малая <3ATR→0.77×, большая→0.24×).
  [[ta-figure-laws-direction-and-extent]]
- **Глубокие факторы fade**: главные — **корот­кая коррекция (+0.61) и рассогласование с мульти-ТФ трендом
  (0/3 ТФ +0.52)**; ОБЪЁМ почти не влияет (debunk «volume confirms»). Мультивариат HistGBM **OOS AUC 0.70
  vs shuffle 0.52** (бьёт shuffle, редкий чистый ML-сигнал; почти весь в corr_piv+mtf_align). [[ta-fade-law-deep-factors]]
- **Применение к ТЕКУЩЕМУ графику**: `apply_current.py` тянет live BTC 1h (Binance), читает последние сетапы
  по законам (fade-направление + сила по факторам + цель ±1.5ATR vs учебная континуация-фольклор).

## Новые pitfalls (known-pitfalls.md)
high-avg-R+низкая частота ≠ standalone-цепочка · грид-ΣR завышен per-top-cap (ранжировать per-trade+null) ·
стена красивых графиков = selection bias + пробой по статич.экстремуму наклонного канала = ложный разворот
(мерить по проекции линий + bias-free fwd-return).
