## ⭐ ТЕКУЩИЙ ЭТАЛОН (что считается лучшим прямо сейчас)

**Эталон предсказания разворотов = подход Андрея (etap_171-173, ветка andrey).**
К нему Андрей пришёл эмпирически, и он совпал с каноном ICT из присланных PDF (sweep/DOL).

**Лучшая нейросеть (ветка pavel):** [[neural-full-arsenal-pavel-etap177]] — AUC **0.91-0.94** на метке Андрея (фрактал + движение ≥X% за 14 баров), 3 актива pooled, 83 фичи (sweep/DOL + зоны + 13 Bulkowski + SADF + frac_diff + entropy). LONG precision до 0.93.

**Ключевые уроки (не забыть):**
1. **Метка решает больше модели:** та же сеть на метке 5%-race → AUC 0.67, на метке Андрея → 0.93.
2. **Нейросеть нужны данные:** на 1 активе (BTC, 3043 строки) застревает на 0.67; на 3 активах (9129) → 0.93.
3. **Нейросеть НЕ лучше LightGBM** на табличных фичах (López de Prado прав).
4. **Высокий WR = подозрение на lookahead** → ВСЕГДА shuffle-тест (mean shuffle-AUC должен быть ~0.5).

## ❌ etap_185: нейросеть НЕ усиливает развороты Вадима (отрицательный результат)

Прогнал нейросеть-арсенал (etap_177) на Core-сигналах ViC Vadim (sweep HTF-зон + maxV).
- Vadim Core: 540 сигналов (BTC/ETH/SOL 2017+), **74% становятся фракталами** (подтверждает vault ~78%), 48% дают ≥5% движение.
- Нейросеть OOS AUC 0.579 — НО **shuffle AUC 0.586 (3 прогона) ВЫШЕ реального!** → нейросеть НИЧЕГО реального не выучила, «улучшение» 48→59% = АРТЕФАКТ (малая выборка 524 + Вадим уже отфильтровал лучшее).

**ВЫВОД:** нейросеть НЕ усиливает Вадима. Сам ViC Vadim (78% precision на фрактал) — сильный сам по себе, нейросеть поверх бесполезна на узкой выборке. Урок: нейросети нужны ДАННЫЕ (524 мало) + не имеет смысла фильтровать уже-отфильтрованное. Скрипт etap_185_vadim_reversals_nn.py.

**ИТОГ всей нейросетевой ветки (честно):** нейросеть полезна для ШИРОКОГО предсказания разворота из сырого контекста (etap_177 AUC 0.93 на ВСЕХ свечах). Но как ФИЛЬТР поверх готовых сигналов (стратегий или Вадима) — не работает (Bayes error / артефакты на малых выборках). Реальная ценность проекта = стратегии с индикаторами (1.1.1 floating 51.9%, 1.1.4+EMA 67%, ViC Vadim 78%), НЕ нейросеть-фильтр.
