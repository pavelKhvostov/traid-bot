# Команда агентов traid-bot

Специализированные агенты с зашитым знанием проекта. Цель — **профессиональная работа без казусов, забытостей и глупостей**. Каждый агент знает свою область досконально и не даёт повторить известные ошибки.

## Состав команды (7 агентов)

### Стратегии и SMC
- **smc-reviewer** — ревью логики стратегий по канону Smart Money Concepts (vault/knowledge/smc/). Триггер: правка detect_*/find_*/scan_* в strategies/.
- **spec-syncer** — ловит рассинхрон кода strategies/*.py ↔ спецификаций vault/knowledge/strategies/*.md перед коммитом.
- **winrate-truth** — хранитель ПРАВИЛЬНЫХ winrate/RR/риск-цифр. Не даёт считать WR на голых зонах без индикаторов (ошибка 51.9%→35%), помнит что entry/SL/TP/RR продуманы через 3 этапа. Триггер: любое заявление про WR/RR/риск/PnL.

### Бэктесты и валидация
- **backtest-auditor** — ловит look-ahead bias в backtest_*.py / optimize_*.py до деплоя.
- **pitfall-checker** — форсит чтение known-pitfalls.md и применение правил до написания кода в чувствительных зонах.
- **ml-validator** — ML/нейросети: ОБЯЗАТЕЛЬНЫЙ shuffle-тест (артефакт vs сигнал), lookahead-чек, знает что работает (разворот AUC 0.93) и что нет (исход = Bayes error ρ 0.09). Триггер: etap_*.py с NN/GBM, заявления про AUC/ρ.

### Инфраструктура
- **bot-vault-guardian** — страж Telegram-бота, vault и git. Не даёт остановить бота, закоммитить токен, сломать дедуп, записать знание не туда. Триггер: работа с ботом/коммитами/.env/vault.

## Как использовать

Агенты триггерятся автоматически по описанию задачи, либо вызывай явно:
> «прогони ml-validator на etap_186», «спроси winrate-truth какой реальный WR 1.1.4», «пусть bot-vault-guardian проверит перед коммитом»

Либо просто работай — Claude сам подключит нужного агента по контексту.

## Зачем это (история)

Агенты созданы после серии казусов: посчитанный WR без индикаторов (занижение 1.1.1 до 35% вместо 51.9%), заявление «риск не продуман» когда всё было зафиксировано через 3 этапа, попытки выдать артефакт (shuffle ≥ real) за результат. Каждый агент закрывает конкретную дыру.
