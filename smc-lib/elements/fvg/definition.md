# FVG (Fair Value Gap)

3-свечный гэп между первой и третьей свечой, образованный displacement-свечой посередине.

## Свечи

c1, c2, c3 — три последовательные свечи. c2 — displacement.

> **Замечание о наименованиях**: внутри FVG используется собственная нумерация c1/c2/c3. При композиции с i-RDRB (для 5-свечной стратегии) маппинг такой: FVG.c1 = i-RDRB.C3, FVG.c2 = i-RDRB.C4, FVG.c3 = i-RDRB.C5.

## Направление

- **LONG FVG** (bullish): `c1.high < c3.low` — гэп вверх
- **SHORT FVG** (bearish): `c1.low > c3.high` — гэп вниз

Условия взаимоисключающие. Если ни одно не выполнено — FVG не образован.

## Зоны

```
long FVG:  zone = [c1.high, c3.low]
short FVG: zone = [c3.high, c1.low]
```

В обоих случаях `zone = (bottom, top)`. Высота FVG = `c3.low - c1.high` для long, `c1.low - c3.high` для short.

## Композиция с i-RDRB

5-свечная стратегия: i-RDRB (C1-C4) + FVG (C3-C4-C5). Если направление FVG совпадает с направлением i-RDRB — паттерн усилен.

## Эталонный пример — SHORT FVG поверх SHORT i-RDRB

BTCUSDT 4h, 2026-05-20 23:00 — 2026-05-21 15:00 MSK:

| Свеча | Время | O | H | L | C |
|---|---|---|---|---|---|
| i-RDRB.C1 | 2026-05-20 23:00 | 77667.49 | 77766.60 | 77226.61 | 77552.23 |
| i-RDRB.C2 | 2026-05-21 03:00 | 77552.24 | 78173.15 | 77525.00 | 78078.72 |
| i-RDRB.C3 / FVG.c1 | 2026-05-21 07:00 | 78078.72 | 78180.01 | 77521.00 | 77889.01 |
| i-RDRB.C4 / FVG.c2 | 2026-05-21 11:00 | 77889.01 | 78200.00 | 77147.15 | 77189.10 |
| FVG.c3 | 2026-05-21 15:00 | 77189.10 | 77402.12 | 76719.47 | 77259.46 |

**Проверка FVG**:
- FVG.c1.low (77521.00) > FVG.c3.high (77402.12) ✓ → SHORT FVG
- zone = [77402.12, 77521.00] (высота 118.88)

i-RDRB направление = SHORT, FVG направление = SHORT → паттерн усилен (continuation).

## Связанные элементы

- `rdrb`, `i_rdrb` — базовые структуры, поверх которых строится 5-свечная FVG-стратегия
